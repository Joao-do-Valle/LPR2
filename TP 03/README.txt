Projeto Pessoa - Swing forms

Como compilar:
1. Abra um terminal.
2. Navegue até a pasta src.
   Exemplo: cd /path/to/PessoaProject/src
3. Compile:
   javac com/example/*.java
4. Execute:
   java com.example.Main

Observações:
- Requer JDK instalado.
- Os formulários são implementados com Swing de forma programática (sem GUI builder).
